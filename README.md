redmine-docker
============================================================

要求条件
------------------------------------------------------------
- docker
- docker-compose

起動方法
------------------------------------------------------------
1. ディレクトリに移動する。

  ``` shell
  cd redmine-docker
  ```

2. 起動する

  ``` shell
  docker-compose up -d
  ```

3. 起動状態を確認する

  ``` shell
  docker-compose ps
  ```
